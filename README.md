#Shot in the Dark

Created for 2015 Summer GI Jam

![Screenshot](Screens/screenshot1.png?raw=true "Optional Title")

## Requirements
* 2-4 Players
* 2-4 Gamepads

##Controls
* Left stick moves
* Right stick aims
* Right bumper fires
